Qui suis-je ?

Mes projets

Voici certains projets que j'ai pu réaliser lors de mes dernières années en informatique, ces projets m'ont fait acquérir et approfondir d'importantes compétences techniques et générales.

Asteroids

Reprise du jeu Asteroids sortie en 1979, réalisé à en dehors de mes projets universitaires avec le langage C++. Ce projet m'a permis d'apprendre à utiliser la bibliothèque graphique SFML et la programmation orientée objet dans ce langage.

C++ SFML

Arbre généalogique

Création d'une application permettant de gérer son arborescence familiale, réalisé en tant que projet de lycée. J'ai pu apprendre à gérer et lier une base de données avec le SGBD SQLite avec les langages Python et SQL.

Python SQLite

Refonte d'un site web

Réalisé avec trois autres de mes camarades pour un projet universitaire. L'objectif était de reprendre un site internet (CASBT), et d'améliorer l'expérience utlisateur tout en gardant la charte graphique du site de base.

HTML/CSS Javascript

Implémentation de graphes

Création de classes et d'interfaces de graphes afin de pouvoir y appliquer les algorithmes de Dijkstra et Bellman. Ce projet m'a permis de mieux appréhender le fonctionnement des interfaces en Java et de réaliser des tests unitaires.

Java

Démineur

Développement du jeu Démineur avec un camarade pour un projet universitaire. Avec le langage Visual Basic.NET, nous avons appris à gérer la programmation évènementielle avec une interface graphique.

Visual Basic.NET
